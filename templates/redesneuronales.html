REDES NEURONALES


Resumen

Los datos para aplicar el algoritmo de redes neuronales fueron recopilados a partir de 1985 hasta el 2019. La mayoria de estos datos se obtuvieron de la página USGS. Una vez recolectados los datos se procedió a realizar el respectivo análisis y procesamiento. Al entrenar el algoritmo de redes neuronales se obtuvo una perdida del 17% aproximadamente. Una vez entrenado el modelo se procedió a realizar la respectiva predicción la cual se detalla en la parte inferior de esta página. Si desea ver los datos usados para la prediccion haga clic aqui.

Explicación

El gráfico nos muestra como estan distribuidos los datos reales y como estan los datos que se predijeron. Los puntos de color verde son la distribucion de los datos reales, mientras que los puntos de color rojo son la distribucion de los datos predichos. Como se puede observar la mayoria de los puntos son coicidentes o están muy cerca de serlos.



Comparación del modelo
Explicación

La gráfica que se observa muestra como se comportan los datos reales versus como se comportan los datos predichos. La linea de color azul indica la distribución de los datos reales, mientras que la linea de color naranja muestra la distribucion de los datos predichos. Se puede observar que hay ciertos picos a los que no llega las predicciones y esto se debe a que hay muestras casi nulas de esos datos por lo que es difícil para la red neuronal LSTM realizar correctamente predicciones en esos picos.
